# SBFlash

SBFlash は Excel を使った暗記カード学習ツールです。

## 出題モード

- 通常出題  
- ランダム出題  
- 逆出題  

これらを組み合わせることで効率的に学習できます。

## プロダクト

- SBFlash Lite
- SBFlash Pro
- SBFlash Challenge
- SBFlash Voice

## ディレクトリ構成

engine : 共通エンジン  
ui : UI  
data : 学習データ  
Lite : Lite版  
Pro : Pro版  
Challenge : チャレンジ版  
Voice : 音声版

## 学習データ

SBKnowledgeData.xlsx を利用します。

英単語、資格試験、専門知識など様々な分野に対応できます。
